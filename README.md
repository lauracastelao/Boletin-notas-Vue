# Boletin-notas con Vue

Requirements:

Design an app that records students' grades. The application must have two views as shown in the Figma design In the view where the qualifications must be listed you have two possible options, choose the one you like the most or start with the first and then try the second option.
Remember that the grade must be numerical but in the qualifications they must be shown according to scale. 


## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur) + [TypeScript Vue Plugin (Volar)](https://marketplace.visualstudio.com/items?itemName=Vue.vscode-typescript-vue-plugin).
